# 🛡️AutoSheild Spam Detection Chrome Extension using Machine Learning

## 📌 Overview

The **Spam Detection Chrome Extension** is a browser extension that automatically detects whether the text on a webpage, email, or user input is **Spam** or **Not Spam** using a Machine Learning model.

The extension extracts text from the current webpage or selected content, sends it to a trained ML model, and instantly displays the prediction along with a confidence score. It is designed to help users identify phishing messages, spam emails, promotional content, and malicious text while browsing.

## 🚀 Features

- Detect Spam and Non-Spam text instantly.
- Machine Learning-based prediction.
- Real-time text extraction from webpages.
- User-friendly Chrome Extension popup.
- Displays prediction confidence score.
- Lightweight and fast execution.
- Secure communication between extension and ML backend.
- Responsive UI with modern design.
- Easy integration with different ML models.
- Supports custom datasets for retraining.
